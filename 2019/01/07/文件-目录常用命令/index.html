文件/目录常用命令

发表于 2019-01-07 | 分类于 Linux

目录与路径

常用命令

  • cd: 进入目录
  • pwd: 显示当前路径
  • mkdir: 创建新目录
  • rmdir 删除一个空目录

文件与目录管理

查看文档与目录列表: ls

复制,删除与移动: cp,rm,mv

查看文档内容

  • cat : 由第一行开始
  • tac : 由最后一行开始
  • nl : 显示内容病显示行号
  • more : 一页一页显示
  • less : 与more类似,不过可以向前翻页
  • head : 只看头几行
  • tail : 只看尾几行
  • od : 以二进制方式读取

直接查看内容

直接查看一个文件内容可以使用cat,tac,nl
